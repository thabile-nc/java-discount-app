# java-discount-app
A simple Java Swing application that calculates a 17% discount.
# Features
Enter an amount
Click "Calculate"
Displays the discounted value
Exit button
# Technologies used
Java
Swing(GUI)
# How to run
1. Compile the program:
   Javac Interest.java
2. Run the program:
   java Interest
